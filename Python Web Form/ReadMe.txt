---------------------
 Mac OS Instructions:
---------------------

 1) Download Python from source and install

 2) set alias (run this command in mac terminal)
    -----------------------
    sudo nano ~/.bash_profile;
    -----------------------

3) Add below code to this file and save it.
    -----------------------
    alias python=python3
    -----------------------  

4) check python version
    -----------------------
    python --version
    -----------------------

5) Install pip by following command
    -----------------------
    pip3 install flask
    -----------------------

6) Open source code directory in terminal

7) run this command to start the server
    -----------------------
	python app.py
	-----------------------

8) Open Web browser and navigate to : http://127.0.0.1:5000
